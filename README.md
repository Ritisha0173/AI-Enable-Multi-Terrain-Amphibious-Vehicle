# AI- Enabled Muti-Terrain Amphibious Vehicle
# Problem Statement

Existing surveillance systems in remote and border areas rely heavily on human patrol or fixed monitoring infrastructure, which is risky, inefficient, and limited by terrain constraints. There is a need for an intelligent system capable of operating autonomously across diverse terrains, ensuring continuous monitoring, obstacle avoidance, and threat detection without risking human lives.

# Objective

To design and develop an AI-based autonomous multi-terrain amphibious vehicle capable of navigating land and water environments, performing real-time surveillance, detecting obstacles, identifying human presence, and sensing metallic objects such as landmines using integrated sensors and embedded systems.

# Novelty

1) Integration of multi-terrain amphibious mobility with AI-based surveillance in a single compact system
2) Combination of landmine detection (inductive sensor) and human detection (vision-based) in one platform
3) Dual-mode operation (autonomous + manual control) for flexibility in critical situations
4) Use of low-cost embedded hardware (ESP32) to achieve real-time intelligent decision-making
5) Servo-based dynamic scanning for 360° obstacle detection and navigation

# Solution

The proposed system is an autonomous amphibious robotic vehicle that integrates sensors, embedded controllers, and a vision-based camera to perform real-time surveillance and navigation. The ESP32 processes sensor inputs from ultrasonic and inductive sensors to detect obstacles and metallic threats, while the camera enables human detection and live video streaming. The system autonomously decides movement using programmed logic and can also be controlled manually through a wireless interface, ensuring efficient operation in remote and hazardous environments.
